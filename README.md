    - 데이터 pre-processing
    - 데이터 기초분석
    - 시계열 그래프를 이용한 패턴분석
    - 전력사용량을 기반으로 가구 클러스터링
    - 3가지 종류의 전력요금제 적용하여 

##### 3. mongoDB_data
  - 목적 : 프로젝트 결과 도출
  - 분석내용
    - 데이터 pre-processing
    - 데이터 기초분석
    - 시계열 그래프를 이용한 패턴분석
    - 전력사용량을 기반으로 가구 클러스터링
